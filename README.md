# Real-Time Task Board Backend

A backend service for a collaborative task board application built with **Node.js and MongoDB**.  
The project focuses on **clean architecture, backend fundamentals, and scalability**.

---

## Features

- Express.js REST API
- MongoDB integration with Mongoose
- Environment-based configuration using dotenv
- Clean layered architecture (routes, controllers, services, models)
- Board creation API
- User registration with secure password hashing
- Git-based development workflow

---

## Tech Stack

- Node.js
- Express.js
- MongoDB (Mongoose)
- bcrypt
- dotenv
- Postman
